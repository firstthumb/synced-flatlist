# React Native Synced Flatlist

Synced Flatlist is created based on the example ( https://youtu.be/gjC2oUJhePE )

### Usage

1. Register new client on [Pexels Developer](https://pexels.com).
2. Create `config.ts` and export `API_KEY`.
3. Type in terminal:

    yarn start
